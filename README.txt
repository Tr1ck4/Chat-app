Guide for set up project 

1. Install the project
2. Run commands:
    cd chat
    npm run build #build frontend components
    npm start     #build backend including database

3. Open browser and go to url http://localhost:3000

You should see the login page first due to authentication.
If you dont have an account, please click on the Register below the login button.
After create the account, you should be redirected to login page.
Now you can login with that account to access homepage.
Feel free to explore next if there are more users are on (use private browser).
